Selected Projects
  1. JOS Operating System [Code]

    Based on skeleton code provided by MIT 6.828, I implemented major parts of JOS in an exokernel style. It consists of booting, memory management, user-level environments, premptive multitasking, file systems and network driver. Also I have read the major source code of xv6 through the process.

  2. Android Memory Managemnt [Code]

    In the first part, I wrote new system calls to map a target process's page table into the current process's address space. Based on that, I implemented VATranslate to get physical address of given virtual address.
    In the second part, I modified the Linux Page Replacement Algorithm by adding new variables reflecting page importance.

  3. Distributed Algorithms [Code]

    Based on skeleton code provided by Princeton COS418 and UW CSE452, I implemented some common distributed algorithms and protocols using Go.

    • A simplified distributed version of MapReduce
    • Chandy Lamport Snapshots
    • Raft Algorithm
    • Paxos Algorithm

  4. NEMU [Code]

    NEMU provides simulated hardware resources to run user program. Based on skeleton code provided by NJU-ICS, I implemented major parts of NEMU in x86 version.
    It mainly includes a small debugger like gdb and some other parts. This project is still in progress
